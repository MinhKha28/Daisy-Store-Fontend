# 🌼 Daisy Frontend — Healthy Store Website

Frontend cho hệ thống thương mại điện tử **Daisy – Healthy Store**, chuyên bán các sản phẩm tốt cho sức khỏe như nước yến, thực phẩm dinh dưỡng, đồ ăn healthy và thức uống tốt cho cơ thể.

Website kết nối với Backend API để thực hiện đăng ký tài khoản, đăng nhập, mua hàng và quản lý người dùng.

---

## 🚀 Tech Stack

### ⚛️ Core

- ReactJS
- TypeScript
- Vite
- React Router DOM

### 🎨 UI

- TailwindCSS
- Responsive Design

### 🔐 Authentication

- JWT Access Token
- Refresh Token (HTTP Only Cookie)
- Axios Interceptor

### ⚡ State Management

- React Query
- Context API
- Custom Hooks

### 💬 Realtime

- Socket.io Client
